Thoughts on Progressive Enhancement

Progressive enhancement is all about the focus of basic content and functionality of the webpage to allow everyone to have access to the content while other users that can afford it, can have the advanced features. The main focus is the pages' text using html, followed by the CSS, and lastly the enhancements provided by JavaScript. Progressive enhancement is important because the focus of the website should be focused on compatibility for every user. Meeting the needs of any user can help the traffic of the website instead of losing a user do to inability to access the site. Website Source
